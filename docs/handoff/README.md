# Handoff

本目录存放接手包索引。接手包不是完整设计文档，而是把某个版本、模块或阶段的关键信息串起来，方便下一个人快速进入状态。

## 命名规则

```text
YYYY-MM-DD-topic.md
```

示例：

```text
2026-01-01-payment-module.md
```

## 建议内容

- 背景和目标
- 已完成内容
- 未完成内容
- 关键代码入口
- 关键文档入口
- 本地运行和验证方式
- 已知风险和踩坑
- 下一步建议
- （前端额外）设计还原对照（设计稿 vs 实现截图）、组件清单（复用 / 自研）、关键页面与各交互态截图
